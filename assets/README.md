# assets/

Place your profile photo here with the name **`photo.jpg`**.

Accepted formats: `.jpg`, `.jpeg`, `.png`, `.webp`

The site loads `assets/photo.jpg` automatically — no extra setup is required.
If you want to use a different path or filename, set `photo` in `cv.js`:

```js
photo: "assets/my-photo.png",
// or an external URL:
photo: "https://github.com/username.png",
```
